Attempts to figure out who was responsible for starting the build and return information about them.

Return Value

A java.util.Map will be returned, with as many of the following properties as could be determined:

PropertyDescription
BUILD_USERThe username of the user that started this build.
BUILD_USER_EMAILThe e-mail address of the user that started this build.
BUILD_USER_IDThe "user id" of the user that started this build.
BUILD_USER_FIRST_NAMEThe first name of the user that started this build.
BUILD_USER_LAST_NAMEThe last name of the user that started this build.
BUILD_USER_GROUPSThe jenkins user groups of the user that started this build.
